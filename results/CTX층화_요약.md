# 계열 길이(가용 이력) 층화 + 호주 ARIMA 보완 — 결과 요약 (2026-07-09)

> 4.5절([재료없음] 해소)·4.6 한계 갱신용. 원자료: ts_{kpx,kepco,aus_elec0}_spread_ctx{168,2048,4096,8192}.json + 기존 ctx512(ts_*_spread.json) + ts_aus_elec.json(arima 추가분)

## A. 조작화 (4.5 서술의 전제)
- 원 계획(다년치 확보 후 캘린더 길이 층화)을 **가용 이력(컨텍스트) 길이 층화**로 조작화: zero-shot 모델에게 "계열 길이"의 실질은 추론 시 활용 가능한 과거 관측의 양(CTX). CTX ∈ {168(1주), 512(3주), 2048(12주), 8192(약 1년; KPX는 계열 제약으로 4096)}.
- 신규 데이터: **KEPCO 전국 시간별 전력사용량**(data.go.kr 15151157, 본부별 합산 → 전국, 2022-01~2024-12 = 26,304h, UTF-8-sig). KPX(수요량)와 기관·측정 정의가 다른 **제2의 국내 계열**. 코퍼스-외부 근거는 KPX와 동일 논리(등급 B/C — GIFT-Eval·Chronos 코퍼스 한국 데이터 0건 확인 승계).
- 프로토콜: spread 8윈도, H=168, 블록 부트스트랩 동일. **KEPCO·호주는 계열이 길어 CTX 변화에도 평가 구간 고정**(snaive 불변: 1.879/1.136) → 깨끗한 통제 비교. KPX(8,760h)는 CTX가 커지면 spread 가용 구간이 축소·이동해 snaive도 변함(1.221~1.583) → 계열 내 비교 교란, 참고용.

## C. 판독 (RQ2 시계열 축의 답)
1. **1주 이력만으로도 우위 존재** — 전 계열·전 모델에서 CTX 168h에 이미 27~46% 우위. FM의 우위는 긴 이력을 전제하지 않는다.
2. **3주(512h)에서 대부분 확보, 이후 수확 체감** — KEPCO(통제): 43~46% → 58~60% → 이후 +0~2%p. 이력을 1년까지 늘려도 우위는 포화. 정형 축의 "소표본 우위 → 대표본 수렴"(5.3)과 대칭적인 포화 구조.
3. **우위 경계는 하한 쪽에 있다** — 정형은 표본이 커지면 GBDT가 따라와 우위가 소멸했지만, 시계열은 이력이 짧을 때 우위가 줄어들 뿐(그래도 양수) 소멸 지점은 이 범위에서 관측되지 않음.
4. 호주 chronos2의 2048 요동(35%)은 단일 계열·8윈도 소표본 변동 범위로 해석(CI 중첩), 단조 주장 금지.

## E. 한계 (4.5·4.6 갱신 시 병기)
- 조작화 주의: 가용 이력 층화는 "모델에 준 과거의 양"이지 "캘린더 상 계열의 존재 기간"이 아님 — 후자(예: 신설 관측 지점)의 함의는 전자로 근사됨을 명시.
- KPX의 비단조(4096에서 하락)는 계열 길이 제약에 따른 평가 구간 이동 교란 — 통제 비교는 KEPCO·호주가 근거.
- KEPCO는 사용량(한전 판매), KPX는 수요량(계통) — 정의 차이로 두 계열은 상호 대체가 아니라 보완.
