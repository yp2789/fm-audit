## C. H-4 재산출 (동일 재실행 배터리)
- 포인트 추정치: 논문 값과 13/14 정확 일치. **예외 1: Moirai KPX tail 0.491→0.490** — Moirai만 샘플 기반 추론이라 ±0.001 지터(재현성 서술 정정 필요: "결정적" → "Moirai는 샘플 중앙값, 3자리에서 ±0.001 변동 가능").
- ACF 블록길이 실측: tail 배치 24~34(FM 대부분 하한 24, snaive KPX 30, 호주 FM 34) / spread 배치 일부 계열 최대 102(aus4 FM 100~102, kepco 69~77) → 해당 CI 소폭 확대. ⚠️정정(2026-07-10 델타검증): 초판의 '24~34'는 tail만 본 과소 범위, ETS 벤치 0.373은 0.372의 오반올림 — 본문 반영 완료. **비중첩 판정 전부 유지** (KPX tail: FM 상한 0.581 < naive 하한 0.884).
- 갱신 대상 CI: KPX naive [0.879,1.353]→[0.884,1.380], ARIMA [0.981,1.259]→[0.981,1.258], 호주 chronos2 [0.944,1.372]→[0.911,1.398], moirai2 [0.949,1.391]→[0.933,1.426]. FM KPX 3종 CI는 불변(blk=24 유지).
