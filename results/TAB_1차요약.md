## 판독 (확정)
1. **RQ1(재현): PFN 계열 전체에서 방향 재현.** TabPFN·TabICL 모두 n=500~1,000에서 최강 GBDT(CatBoost) 대비 우위 — n=500은 두 모델 다 CI 비중첩. "소표본 우위"가 단일 모델이 아닌 **계열 차원**에서 코퍼스-외부 국내 데이터에 재현됨.
2. **RQ2(경계): 우위 소멸 경계 n≈2,000~5,000.** TabICL은 n≈2,000부터, TabPFN은 n≈5,000부터 CatBoost와 CI 중첩 동률 진입. n=10,000에서 상위 3개(TabPFN 0.914·CatBoost 0.913·TabICL 0.910)가 사실상 동률 — "대표본에서 GBDT 추격" 문헌 패턴(TabPFN-2.5 카드·Grinsztajn 통설)과 정합. TabPFN은 대표본에서도 붕괴하지 않고 동률 유지(명목 1위).
3. **비용 비대칭**: 동률 구간(n=10,000)에서 TabPFN 745s vs CatBoost 15s — **약 50배**. 성능 동률·비용 비대칭은 실무 모델 선정의 핵심 함의(7.1 재료).
4. **GBDT 내부 순위**: CatBoost > LightGBM > XGBoost 전 구간 — 범주형 지배적 데이터에서 CatBoost 강세 통설 재현.

## 감사 관점 특기사항
- **TabPFN 접근 게이트(7장 재료, 실측 기록)**: 가중치 다운로드에 ① Prior Labs 계정 가입 ② 버전별 라이선스 개별 동의(2.5/2.6/3 각각) ③ API 키 발급·환경변수 설정이 필요했고, 진단 과정에서 토큰 유효/라이선스 미동의 상태 구분에 API 직접 조회가 필요했음. 추가로 CPU에서 n>1000은 기본 차단되어 공식 플래그(ignore_pretraining_limits)로 해제해야 했음 — 제3자 감사의 실무 장벽 실증 사례. 코퍼스 투명성(2.4)과 별개의 "감사 가능성" 차원.
- 코퍼스-외부 근거: 2026-06 갱신분은 TabPFNv2(2025-01)·TabICL(2025-02) 공개 이후 생성 데이터 포함. 단 PFN 계열은 합성 사전학습이므로 이 축의 감사 질문은 "합성 prior가 국내 실데이터 분포를 커버하는가"로 번역됨(3.5.2 C등급 상한 준수).
